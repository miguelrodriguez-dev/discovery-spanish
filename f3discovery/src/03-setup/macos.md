# macOS

Todas las herramientas se pueden instalar usando [Homebrew]:

[Homebrew]: http://brew.sh/

``` console
$ # ARM GCC debugger
$ brew install arm-none-eabi-gdb

$ # Minicom and OpenOCD
$ brew install minicom openocd
```
